The Silverlight Toolkit contains the following items:

  • User License
  • Silverlight Libraries - Silverlight 4 Libraries included in the Toolkit. You will also find these in your Visual Studio toolbox when creating Silverlight applications.
  • Documentation - Documentation for the Silverlight Toolkit controls in this release.
  • Samples - Sample browser containing all of the controls, sample source usage, and useful scenario examples.
  • Themes - Professional themes for your applications, including XAML source.
  • Source - Source code for the samples is included. If selected during setup, the Silverlight Toolkit's source code is also included in this location. Copy the source Zip file(s) to your desktop, or other user folder, and extract the contents to get started.
  • Silverlight Development and Infrastructure Tools - including Visual Studio project templates and more.

Web resources:

  • Silverlight.net - A community site dedicated to Silverlight development that includes blogs, forums, webcasts, whitepapers, and more.
  • Silverlight.net Discussions - User forums for discussing the Silverlight Toolkit and other controls.
  • Silverlight Toolkit on CodePlex - The official Silverlight Toolkit release web site. Also includes the Issue Tracker used for opening issues and feature requests.
This release is designed for Silverlight 4 application development. Support for Silverlight 3 is available in previous releases.
